# Project Logos

Platziere hier die Logos deiner Projekte.

## Benötigte Dateien:

- `clubmatch-logo.png`
- `bravodesk-logo.png`
- `taplock-logo.png`
- `adressverwaltung-logo.png`

## Empfohlene Größe:
- Mindestens 200x200px
- Quadratisches Format (1:1 Ratio)
- PNG oder JPG Format
- Transparenter Hintergrund empfohlen (PNG)

Die Logos werden automatisch auf 120x120px skaliert und zentriert.
